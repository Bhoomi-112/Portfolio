A modern, responsive portfolio website showcasing AI/ML projects and skills with smooth animations and interactive elements.

## 🌟 Features

- **🐾 Cat-Themed Loader**: A cute cat animation that greets users while the page loads!
- **✨ Particle Universe**: Interactive background that responds to your touch and mouse.
- **🚀 Ultra-Smooth Scroll**: High-performance scrolling with scroll-triggered visual effects.
- **🌈 Dynamic Gradients**: Modern dark theme with vibrant, pulsing green accents.
- **📱 Fluid Design**: Seamlessly shifts between desktop, tablet, and mobile magic.
